# Galería Interactiva con JavaScript

Proyecto desarrollado para aplicar manipulación del DOM y manejo de eventos con JavaScript.

## Funcionalidades
- Agregar imágenes mediante URL
- Seleccionar una imagen (resaltada)
- Eliminar la imagen seleccionada
- Diseño responsive con CSS Grid

## Tecnologías
- HTML5
- CSS3
- JavaScript
- GitHub Pages
